## What's inside

**Pipeline** — Shared deal tracker with referral source, commission flags, and 5-touch sequence engine

**Register Lead** — 10-field Ramp registration with conflict detection and Slack message generator

**Templates** — 24 Tarun templates + 13 Ed templates, voice-matched, EAP and commission flags built in

**Sage Clarifier** — Instant ERP identification, live/EAP/commission status, clarifying questions

**Win Wire Generator** — Upload a Teams transcript or fill fields → generates PPTX for Maddie automatically

**Writing Assistant** — 12-framework library (Voss, Blount, Welsh, Hormozi, Aaron Ross, Ogilvy, Van Edwards, Cialdini, Carnegie, Kahneman, Pink, Navarro) — Tarun only

**AI Secretary** — Proactive action queue, urgency-sorted, flags missed follow-ups and overdue win wires

**Houston Networking** — 7 finance groups (FEI, CFO Leadership Council, TXCPA, CFO Focus, GHP, Houston West, A&F Meetup) with ROI calculator and event tracker

**Quick Capture** — In-person lead logging from networking events, syncs to Google Sheets via Slack bot

**Dan's Dashboard** — Commission tracker, referral source management, sign-off queue, Slack alert previews

## Tech Stack

- Single HTML file — no backend, no database, no dependencies
- Hosted on GitHub Pages (free, permanent URL)
- Make.com automates Google Sheets sync, Slack notifications, Outlook emails
- Win wire PPTX generated by `winwire_generator.py`

---

## Updating the dashboard

1. Download the latest `kerr-ramp-sage-final.html` from Claude
2. Rename it to `index.html`
3. Go to this repo → click `index.html` → pencil ✏️ icon → replace file → upload → Commit
4. GitHub Pages rebuilds in ~60 seconds
5. Ed and Dan refresh — they see the update instantly
